C3 Relation 計數公式：9.1 #25、#47、#49

這一節不要死背公式。正確做法是先把 A×A 拆成兩塊： diagonal pairs 和 off-diagonal pairs。所有公式都只是這兩塊的乘法結果。

核心分解：先分兩個區域

1. Diagonal pairs

形式是 (a,a)，也就是元素跟自己配對。

如果集合有 n 個元素，就剛好有 n 個 diagonal pairs。

例如 A={1,2,3} 時，diagonal pairs 是：

(1,1),(2,2),(3,3)

它們負責判斷 reflexive、irreflexive、asymmetric。

2. Off-diagonal unordered groups

先用無順序組 {a,b} 分組，其中 a≠b。

這樣的組數是：

C(n,2)=n(n−1)/2

每一組 {a,b} 對應兩個 ordered pairs：

(a,b) 和 (b,a)

它們負責判斷 symmetric、antisymmetric、asymmetric。

小心：C(n,2) 是在數「不同元素的無順序組」； n(n−1) 是在數「不同元素的 ordered pairs」。 這兩個不能混用。

逐行推導：每一行怎麼來

1. 任意 relation on n elements：為什麼是 2^{n²}？

relation on A 的意思是：

R⊆A×A

如果 |A|=n，那 A×A 有：

n·n=n² 個 ordered pairs

每一個 ordered pair 都只有兩個選擇：放進 relation，或不放進 relation。

物件選擇數理由
每個 ordered pair2放 / 不放
共有 ordered pairsn²|A×A|=n²
總數2^{n²}乘法原理：2 乘 n² 次

對應 9.1 #47(a)：若 n=4，總數就是 2^{16}=65536。

2. symmetric：為什麼是 2^{n(n+1)/2}？

symmetric 要求：

(a,b)∈R ⇒ (b,a)∈R

先看 diagonal：(a,a) 的反方向還是自己，所以每個 diagonal pair 可以自由放或不放。

diagonal 選法：2^n

再看每一組不同元素 {a,b}。它有兩個方向：(a,b)、(b,a)。為了對稱，只能：

選法(a,b)(b,a)是否 symmetric
都不放00可以
只放 (a,b)10不行
只放 (b,a)01不行
雙向都放11可以

所以每組只有 2 種選法。這樣的組共有 C(n,2)=n(n−1)/2 組。

2^n · 2^{n(n−1)/2}=2^{n+n(n−1)/2}=2^{n(n+1)/2}

↩ 回公式表

3. antisymmetric：為什麼是 2^n3^{n(n−1)/2}？

antisymmetric 要求：

(a,b)∈R 且 (b,a)∈R ⇒ a=b

白話：不同元素不能雙向都有。注意 diagonal (a,a) 沒問題，因為這裡 a=b。

所以 diagonal 仍然自由放或不放：

diagonal 選法：2^n

每一組 {a,b}，a≠b，有四種原始選法，但「雙向都放」會違反 antisymmetric：

選法(a,b)(b,a)是否 antisymmetric
都不放00可以
只放 (a,b)10可以
只放 (b,a)01可以
雙向都放11不行，因為 a≠b

所以每組有 3 種。共有 C(n,2)=n(n−1)/2 組。

2^n · 3^{n(n−1)/2}

4. asymmetric：為什麼是 3^{n(n−1)/2}？

asymmetric 要求：

(a,b)∈R ⇒ (b,a)∉R

它比 antisymmetric 更嚴格。先看 diagonal：如果放了 (a,a)，那它的反方向還是 (a,a)，會直接違反 asymmetric。

diagonal 選法：全部不能放，所以只有 1 種

每組不同元素 {a,b} 的選法和 antisymmetric 一樣，不能雙向都有，所以有 3 種：

都不放、只放 (a,b)、只放 (b,a)

共有 C(n,2)=n(n−1)/2 組，因此：

1·3^{n(n−1)/2}=3^{n(n−1)/2}
考場提醒：asymmetric 一定 irreflexive；所以 diagonal 不能有任何一個 1。

5. irreflexive：為什麼是 2^{n(n−1)}？

irreflexive 要求所有 diagonal pairs 都不能放：

(a,a)∉R for every a∈A

所以 diagonal 沒有自由度：全部固定不放，只有 1 種選法。

剩下的是 off-diagonal ordered pairs，也就是 (a,b) 且 a≠b。

第一格 a第二格 b總數
n 種不能等於 a，所以 n−1 種n(n−1)

每個 off-diagonal ordered pair 都可以自由放或不放，所以：

2^{n(n−1)}

也可以用分組看：每組 {a,b} 有 4 種選法，共 C(n,2) 組，所以 4^{C(n,2)}=2^{2C(n,2)}=2^{n(n−1)}。

6. reflexive and symmetric：為什麼是 2^{n(n−1)/2}？

同時滿足 reflexive 和 symmetric。

reflexive 先固定 diagonal：所有 (a,a) 都必須放。這部分沒有自由度，只有 1 種選法。

diagonal 選法：全部必放，1 種

symmetric 再限制每組 {a,b}：只能「都不放」或「雙向都放」。

每組 2 種，共 C(n,2)=n(n−1)/2 組

所以：

1·2^{n(n−1)/2}=2^{n(n−1)/2}

7. neither reflexive nor irreflexive：為什麼是 (2^n−2)2^{n(n−1)}？

這句話最容易被誤解。它不是「不是 reflexive 就好」，而是同時：

不是 reflexive，而且也不是 irreflexive
狀態diagonal 情況
reflexive全部 diagonal 都放
irreflexive全部 diagonal 都不放
neither reflexive nor irreflexive有些 diagonal 放，有些 diagonal 不放

diagonal 共有 n 個，每個本來可放可不放，所以總選法是 2^n。

但要扣掉兩種不合法情況：

  • 全部都放：這會變成 reflexive。
  • 全部都不放：這會變成 irreflexive。
diagonal 選法：2^n−2

off-diagonal 完全沒有額外限制，每個 ordered pair 都自由放或不放。off-diagonal ordered pairs 有 n(n−1) 個，所以：

off-diagonal 選法：2^{n(n−1)}

兩部分相乘：

(2^n−2)2^{n(n−1)}

(a,c)∈S∘R iff 存在 b，使 (a,b)∈R 且 (b,c)∈S

重點：S∘R 是「先走 R，再走 S」。

符號意思圖上怎麼想
R²R∘R走 2 步能到
R³R²∘R走 3 步能到

9.1 #37 警告：此題答案本對某些 composition 結果列成 R₂，但依課本 composition 定義，用「存在中繼點」重算，部分小題會得到 universal relation。考試前請拿此題向老師確認採用哪個答案；其他題型不受影響。
展開：9.1 #37 的安全解法

寫成「存在 b 同時滿足兩個不等式」，再判斷 a,c 之間被迫有什麼關係。如果不管 a,c 如何都能選到 b，答案就是 universal relation ℝ×ℝ。

C8 Partial order / Hasse / Lattice：9.6

Partial order = reflexive + antisymmetric + transitive
名詞意思不要搞錯
poset集合 + partial ordering不是每對都要能比較。
comparablea≤b 或 b≤a至少一個方向成立。
incomparable兩個方向都不成立常出在 subset 或 divides。
maximal沒有人在它上面可以有很多個。
greatest它在所有人上面最多一個；不一定存在。
minimal沒有人在它下面可以有很多個。
least它在所有人下面最多一個；不一定存在。
upper bound同時在指定元素上面整除 poset 中像共同倍數。
least upper bound所有共同上界中最低者整除 poset 中常像 lcm。
lower bound同時在指定元素下面整除 poset 中像共同因數。
greatest lower bound所有共同下界中最高者整除 poset 中常像 gcd。
lattice任兩元素都有 LUB 和 GLB只要一對失敗就不是。
