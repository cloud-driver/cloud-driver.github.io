C1 Relation 基本定義

Binary relation from A to B：R⊆A×B
Relation on A：R⊆A×A

How many relations?問有多少 subsets先算 A×B 有幾個元素，再做 2^m。

C2 Relation 性質總表

性質定義矩陣看法有向圖看法一句話口訣
reflexive所有 (a,a) 都在主對角線全 1每個點都有 loop每個人都指自己
irreflexive所有 (a,a) 都不在主對角線全 0沒有任何 loop沒有人指自己
symmetric(a,b)⇒(b,a)M=Mᵀ箭頭都要雙向有來就有回
antisymmetric(a,b) 且 (b,a) ⇒ a=b非對角線不可雙向都是 1不同點不能互指雙向只准發生在自己
asymmetric(a,b)⇒not (b,a)非對角不可雙向；主對角全 0不能有 loop，也不能雙向比 antisymmetric 更嚴
transitive(a,b),(b,c)⇒(a,c)M⊙M ≤ M兩段路要有捷徑接得起來就要補到
高頻錯誤：antisymmetric 不是 symmetric 的相反。identity relation 同時 symmetric 和 antisymmetric。

C3 Relation 計數公式：9.1 #25、#47、#49

核心分解

Diagonal pairs

(a,a)，共有 n 個。

負責 reflexive / irreflexive / asymmetric 的判斷。

Off-diagonal unordered groups

每組 {a,b}，a≠b，共有 C(n,2) 組。

每組對應兩個方向：(a,b),(b,a)。

條件diagonal 選法每組 {a,b} 選法總數
任意 relation on n elements自由四種：都不放、只放一邊、只放另一邊、雙向2^{n²}
symmetric2^n2 種：都不放 / 雙向都放2^{n(n+1)/2}
antisymmetric2^n3 種：都不放 / 只放一邊 / 只放另一邊2^n 3^{n(n−1)/2}
asymmetric全部不能放3 種3^{n(n−1)/2}
irreflexive全部不能放每個 ordered pair 自由2^{n(n−1)}
reflexive and symmetric全部必放2 種2^{n(n−1)/2}
neither reflexive nor irreflexive有些放有些不放：2^n−2每個 ordered pair 自由(2^n−2)2^{n(n−1)}

C4 Composition / Power：9.1 #37、9.3 #15、9.4 #19

(a,c)∈S∘R iff 存在 b，使 (a,b)∈R 且 (b,c)∈S

重點：S∘R 是「先走 R，再走 S」。

符號意思圖上怎麼想
R²R∘R走 2 步能到
R³R²∘R走 3 步能到
R*R∪R²∪R³∪…走任意正步數能到
9.1 #37 警告：此題答案本對某些 composition 結果列成 R₂，但依課本 composition 定義，用「存在中繼點」重算，部分小題會得到 universal relation。考試前請拿此題向老師確認採用哪個答案；其他題型不受影響。
展開：9.1 #37 的安全解法

寫成「存在 b 同時滿足兩個不等式」，再判斷 a,c 之間被迫有什麼關係。如果不管 a,c 如何都能選到 b，答案就是 universal relation ℝ×ℝ。

C6 Closure：9.4

Closure 就是「最少補 pair，使 relation 具有指定性質」。

Closure要補什麼公式圖上做法
reflexive closure所有缺的 (a,a)R∪Δ每個點加 loop
symmetric closure所有反方向 (b,a)R∪R^{-1}每條箭頭補反向
transitive closure所有 path 可達的 pairR*=R∪R²∪R³∪…有路就補直接關係
高頻錯誤：symmetric closure 不會自動補 loop；transitive closure 不是只補一次。

C8 Partial order / Hasse / Lattice：9.6

Partial order = reflexive + antisymmetric + transitive
名詞意思不要搞錯
poset集合 + partial ordering不是每對都要能比較。
comparablea≤b 或 b≤a至少一個方向成立。
incomparable兩個方向都不成立常出在 subset 或 divides。
maximal沒有人在它上面可以有很多個。
greatest它在所有人上面最多一個；不一定存在。
minimal沒有人在它下面可以有很多個。
least它在所有人下面最多一個；不一定存在。
upper bound同時在指定元素上面整除 poset 中像共同倍數。
least upper bound所有共同上界中最低者整除 poset 中常像 lcm。
lower bound同時在指定元素下面整除 poset 中像共同因數。
greatest lower bound所有共同下界中最高者整除 poset 中常像 gcd。
lattice任兩元素都有 LUB 和 GLB只要一對失敗就不是。

Hasse diagram 三刪一省

步驟動作理由
刪 loop不畫 (a,a)reflexive 已內建。
刪 transitive 邊若 a，不畫 a可以由路徑推出。
省箭頭邊往上畫預設越上越大。
